🚀 DevFinder - GitHub User Profile Explorer

DevFinder is a modern web application that allows users to search and explore GitHub profiles in a clean and interactive UI.

It integrates the GitHub REST API to fetch real-time user data and displays it using dynamic UI components and charts.

🌟 Features

🔎 Core Features
- Search any GitHub username
- Display profile details (avatar, name, bio, stats)
- Show public repositories
- Dark / Light theme toggle
- Responsive modern UI

📊 Advanced Features
- Language usage chart (Chart.js)
- Top 5 repositories sorted by stars
- Compare two GitHub users side-by-side
- Recent searches saved using localStorage
- Export profile as PDF

🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
- Chart.js
- html2canvas
- jsPDF

⚙️ How to Run Locally

1. Clone the repository
2. Open the project folder
3. Run using Live Server or localhost
4. Search any GitHub username
